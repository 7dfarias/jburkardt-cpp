FSU_DAKOTA
Additions to the Sandia DAKOTA Package

FSU_DAKOTA is a collection of C++ libraries and programs which were developed for inclusion with the DAKOTA package that is maintained at Sandia National Laboratory.

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Related Data and Programs:

CVT, a C++ library which computes Centroidal Voronoi Tessellations.

HALTON, a C++ library which computes Halton sequences.

HAMMERSLEY, a C++ library which computes Hammersley sequences.

LATINIZE, a C++ program which can "Latinize" a set of points.

QUALITY, a C++ library which computes the quality of dispersion of a set of points.

Source Code:

FSU.HPP is an INCLUDE file containing the definitions of all the library routines.

  • fsu.hpp, the source code;

FSU_DAKOTA.SH is a shell script that runs all the lower shell scripts that install and test all the components.

  • fsu_dakota.sh, the main shell script;

FSU_SUB is a support library of commonly used routines:

  • fsu_sub.cpp, the library source code;
  • fsu_sub.sh, commands to compile the source code;

FSU_HALTON is a library of routines for computing the Halton sequence:

  • fsu_halton.cpp, the library source code;
  • fsu_halton.sh, commands to compile the source code;
  • fsu_halton_standalone.cpp, an interactive executable program that can be used as a frontend to access and test the library;
  • fsu_halton_standalone.sh, commands to compile the source code;

FSU_HAMMERSLEY is a library of routines for computing the Hammersley sequence:

  • fsu_hammersley.cpp, the library source code;
  • fsu_hammersley.sh, commands to compile the source code;
  • fsu_hammersley_standalone.cpp, an interactive executable program that can be used as a frontend to access and test the library;
  • fsu_hammersley_standalone.sh, commands to compile the source code;

FSU_CVT is a library of routines for computing a set of points via the Centroidal Voronoi Tessellation:

  • fsu_cvt.cpp, the library source code;
  • fsu_cvt.sh, commands to compile the source code;
  • fsu_cvt_problems.cpp, a noninteractive executable program that can be used to test the library;
  • fsu_cvt_problems.sh, commands to compile the source code;
  • fsu_cvt_problems_output.txt, the output from a run of the program;
  • fsu_cvt_standalone.cpp, an interactive executable program that can be used as a frontend to access and test the library;
  • fsu_cvt_standalone.sh, commands to compile the source code;

FSU_LATINIZE is a library of routines which modify a given dataset so that its values form a Latin Hypersquare:

  • fsu_latinize.cpp, the library source code;
  • fsu_latinize.sh, commands to compile the source code;
  • fsu_latinize_problems.cpp, a noninteractive executable program that can be used to test the library;
  • fsu_latinize_problems.sh, commands to compile the source code;
  • fsu_latinize_problems_output.txt, the output from a run of the program;
  • fsu_latinize_standalone.cpp, an interactive executable program that can be used as a frontend to access and test the library;
  • fsu_latinize_standalone.sh, commands to compile the source code;

FSU_QUALITY is a library of routines evaluate various distribution quality measures for a given pointset:

  • fsu_quality.cpp, the library source code;
  • fsu_quality.sh, commands to compile the source code;
  • fsu_quality_problems.cpp, a noninteractive executable program that can be used to test the library;
  • fsu_quality_problems.sh, commands to compile the source code;
  • fsu_quality_problems_output.txt, the output from a run of the program;
  • fsu_quality_standalone.cpp, an interactive executable program that can be used as a frontend to access and test the library;
  • fsu_quality_standalone.sh, commands to compile the source code;
