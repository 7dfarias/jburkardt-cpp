SPHERE_MONTE_CARLO
Monte Carlo Estimate of Sphere Surface Integrals

SPHERE_MONTE_CARLO is a C++ library which estimates the integral over the surface of the unit sphere of a function F(X,Y,Z).

Licensing:

The computer code and data files described and made available on this web page are distributed under the GNU LGPL license.

Languages:

SPHERE_MONTE_CARLO is available in a C++ version and a FORTRAN90 version and a MATLAB version.

Related Data and Programs:

BALL_VOLUME_MONTE_CARLO, a C++ program which applies a Monte Carlo method to estimate the volume of the unit ball in M dimensions;

SPHERE_EXACTNESS, a C++ program which tests the polynomial exactness of a quadrature rule for the unit sphere;

SPHERE_GRID, a C++ library which provides a number of ways of generating grids of points, or of points and lines, or of points and lines and faces, over the unit sphere.

SPHERE_LEBEDEV_RULE, a C++ library which computes Lebedev quadrature rules for the unit sphere;

SPHERE_QUAD, a C++ library which approximates an integral over the surface of the unit sphere by applying a triangulation to the surface;

SPHERE_TRIANGLE_QUAD, a C++ library which estimates the integral of a function over a spherical triangle.

STROUD, a C++ library which defines quadrature rules for various geometric shapes.

TETRAHEDRON_MONTE_CARLO, a C++ program which uses the Monte Carlo method to estimate integrals over a tetrahedron.

TRIANGLE_MONTE_CARLO, a C++ program which uses the Monte Carlo method to estimate integrals over a triangle.
